# Back-End E-Commerce 

## Description
* Backend for an e-commerce site 

## Tools Used to Build
* dotenv
* Express
* Insomnia
* JavaScript
* MySQL
* MySQL2.js
* Node.js


## Instructions for Installation
* npm install
* use env to login to SQL
* mysql source db/schema.sqll
* npm run seed
* npm start
